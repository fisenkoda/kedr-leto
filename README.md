# Лето в «Кедре» — лендинг летних семейных заездов 2026

Лендинг санатория «Кедр» (г. Саянск, Иркутская обл.).
Статический сайт: HTML + CSS + JS, без сборщиков и фреймворков.

## Структура
```
index.html      — страница лендинга
privacy.html    — политика обработки персональных данных (152-ФЗ)
styles.css      — стили
script.js       — логика (карусель, меню, форма, капча, баннер, анимации)
CNAME           — домен (letokurortkedr.ru) для GitHub Pages
assets/
  img/          — изображения (hero, факторы, номера, детям, логотипы партнёров)
  docs/         — PDF (лицензии, памятка, правила, реквизиты)
```

## Интеграции
- **Яндекс.Метрика** — счётчик ID `109685901` (в `<head>`, webvisor/clickmap включены).
- **Яндекс SmartCaptcha** — защита формы. Виджет рендерится из `script.js`
  (client-ключ в коде). Токен проверяется на сервере в Google Apps Script
  (server-ключ хранится только в скрипте) — см. `apps-script-kedr.gs`.
- **Форма заявки** — отправка в Google Таблицу через Apps Script (POST, text/plain).
- **Баннер cookie/ПДн** — нижняя плашка, согласие сохраняется в `localStorage`,
  ведёт на `privacy.html`.

## Анимации
Плавное появление блоков при скролле + всплытие контента hero при загрузке.
Уважают системную настройку «уменьшить движение» (prefers-reduced-motion).

## Деплой
GitHub Pages + домен с reg.ru.
- HTTPS: включён «Enforce HTTPS» в настройках Pages.
- www → без www: через DNS (A-записи apex + CNAME `www` → `<логин>.github.io`).
- `.htaccess`/PHP на GitHub Pages не работают — редиректы и год делаются
  средствами GitHub Pages + DNS + JS.

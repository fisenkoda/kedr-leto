## Как запустить локально

### Вариант А — двойной клик
Просто открой `index.html` двойным кликом в проводнике. Откроется в браузере.

### Вариант Б — VS Code + Live Server (рекомендую)
1. Открой папку `landing` в VS Code: `File → Open Folder`
2. Установи расширение **Live Server** (автор Ritwick Dey):
   - Слева на панели: `Extensions` (значок квадратиков)
   - В поиске набери `Live Server`
   - Установи первый результат (с 50М+ установок)
3. Открой `index.html` в VS Code
4. Внизу справа в статус-баре появится кнопка **«Go Live»** → клик
5. Откроется браузер по адресу `http://127.0.0.1:5500/index.html`
6. Теперь при каждом сохранении файла (Ctrl+S) страница автоматически перезагружается
